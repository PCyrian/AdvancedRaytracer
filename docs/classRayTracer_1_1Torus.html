Detailed Description

3D primitives representating a Torus.

A Torus with a big radius (majorRadius) and a little one (minorRadius) Center with the variable center, the intersexction is numerical calculate

Attempts a ray–torus intersection by numerically solving F(P(t))=0.

Scans the interval [tMinSearch, tMaxSearch] in fixed steps, looks for a sign change in torusField(), and then refines the root by bisection.

Parameters
ORay origin in torus-local space.
DNormalized ray direction.
RTorus major radius (center-to-tube-center).
rTorus minor radius (tube radius).
tMinSearchLower bound of t to scan.
tMaxSearchUpper bound of t to scan.
[out]tHitOn success, the t-value of the intersection.
Returns
true if an intersection was found and tHit set.

Evaluates the torus’s implicit surface function F(P(t)).

F(P) = (x² + y² + z² + R² − r²)² − 4R²·(x² + z²), where P = O + tD.

Parameters
ORay origin in torus-local coordinates.
DNormalized ray direction.
tRay parameter (distance along D).
majorRTorus major radius (center -> tube center).
minorRTorus minor radius (tube radius).
Returns
The value F(P(t)); zero when P lies on the torus surface.
