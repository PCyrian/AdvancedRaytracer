Detailed Description

Infinite, single-sided plane primitive.

Defined by a world-space anchor point and a unit normal vector. Intersections occur only when the ray approaches from the “front” side (i.e. the direction of the normal).

Ray–plane intersection.

Performs a front-face test (denom = N⋅D < 0), then computes t = (P₀−O)⋅N / (D⋅N). Fills HitRecord with point, normal, UV coordinates (scaled and wrapped), and material.
