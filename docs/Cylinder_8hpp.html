Detailed Description

Declares the finite right‐circular Cylinder primitive.

Author
Cyrian Pittaluga
Version
1.0
Date
2025-05

A capped cylinder defined by a base center point, an axis direction, a radius and a finite height. Ray‐intersection handles both the curved side and the two circular end caps, and computes UV coordinates.
