Detailed Description

Implicit tangleCube fractal surface.

  1. Calculate the fourth power : sum4 = x⁴ + y⁴ + z⁴
  2. Calculate the sum of the squares and multiply it by α : sum2 = x² + y² + z² ⇒ α · sum2
  3. Subtract this result from sum4 and add β : F(x,y,z) = sum4 − α·sum2 + β

Implements a ray‐implicit intersection by solving the quartic equation.

Solves a quartic equation Ax⁴ + Bx³ + Cx² + Dx + E = 0.
