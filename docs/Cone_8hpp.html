Detailed Description

Declares the finite (capped) right‐circular cone primitive.

Author
Cyrian Pittaluga
Version
1.0
Date
2025-05

A cone defined by its apex point, a unit‐length axis direction, an apex half‐angle, and a finite height. Supports intersection against both the conical side and the circular base cap, and computes UV coordinates for simple texturing.
